# 停车场管理系统 java89

## 项目概述

停车场管理系统是一款基于前后端分离架构的Web应用，旨在为用户提供便捷的停车服务与管理功能。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis-Plus
  - Vue

## 功能模块

### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理

## 系统角色

- 普通用户
- 管理员

## 运行环境

- Windows

## 目录结构

```
 ├── frontend  // 前端源代码
 ├── backend   // 后端源代码
 ├── document  // 项目文档
 └── resources // 项目资源
```
